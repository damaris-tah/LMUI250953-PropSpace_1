# PropSpace Full-Stack App

PropSpace is a full-stack property listing application built with **React (frontend)** and **Node.js/Express + MongoDB (backend)**.  
It includes authentication, CRUD operations, and a dashboard for managing properties.

---

## 🚀 Features
- User registration and login (JWT authentication)
- Add, update, delete properties
- View public and private property feeds
- Dashboard with summary cards, transactions, and notifications
- Responsive React frontend with routing
